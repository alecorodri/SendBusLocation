## Installation Step

1. Clone the repository.

 ```
git clone https://github.com/alecorodri/SendBusLocation.git
 ```

2. Open the project with Visual Studio Code/Android Studio.

3. Open pubspec.yaml file.

4. Run in project for change packages

```
flutter pub get
```
